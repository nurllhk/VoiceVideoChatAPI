# package usage

```sh 
 ZegoUIKitPrebuiltCall(
              appID: DefaultKey.AppId,
              appSign: DefaultKey.Api,
              callID: callid(),
              userID: uid,
              userName: "user_$uid",
              config: ZegoUIKitPrebuiltCallConfig.oneOnOneVideoCall()
                ..onOnlySelfInRoom = (context) => Navigator.pop(context))

 ```
